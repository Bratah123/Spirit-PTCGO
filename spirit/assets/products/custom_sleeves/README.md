# Custom Sleeves

Place your custom card sleeve PNG images in this directory.

### Specifications
* **Recommended Resolution:** **`512x512`** pixels (standard for high-fidelity original sleeves) or **`256x256`** pixels (standard for set sleeves).
* **Aspect Ratio:** `1:1` (Square).
* **File Format:** PNG (Alpha channel transparency is supported).
* **Naming:** Use lowercase letters, numbers, and underscores (e.g., `my_custom_sleeve.png`). The filename (without extension) becomes the logical asset ID.

*Note: The compilation tool will automatically resize any non-standard PNGs to fit the template slot dimensions.*
