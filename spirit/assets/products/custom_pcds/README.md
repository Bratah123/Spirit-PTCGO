# Custom Theme Decks (PCDs)

Place your custom PCD/theme deck box flat PNG textures in this directory.

### Specifications
* **Recommended Resolution:** **`512x512`** pixels (standard for all official PCD box flats).
* **Aspect Ratio:** `1:1` (Square).
* **File Format:** PNG.
* **Naming:** Use lowercase letters, numbers, and underscores (e.g., `dark_hammer_deckbox.png`). The filename (without extension) becomes the logical asset ID.

*Note: This texture represents the flat box skin used by the client's PCD renderer. The compilation tool will automatically scale any non-standard PNGs to `512x512` on server boot.*
