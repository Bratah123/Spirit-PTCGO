# Custom Coins

Place your custom gameplay coin PNG textures in this directory.

### Specifications
* **Recommended Resolution:** **`256x256`** pixels (exact standard for all official PTCGO gameplay coins).
* **Aspect Ratio:** `1:1` (Square).
* **File Format:** PNG (Transparency should be used outside the coin's circular boundary to keep it round).
* **Naming:** Use lowercase letters, numbers, and underscores (e.g., `my_custom_coin.png`). The filename (without extension) becomes the logical asset ID.

*Note: The compilation tool will automatically scale non-standard PNGs to `256x256`.*
