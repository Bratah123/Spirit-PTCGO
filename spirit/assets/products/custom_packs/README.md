# Custom Booster Packs

Place your custom booster pack PNG textures in this directory.

### Specifications
* **Recommended Resolution:** **`256x395`** or **`512x790`** pixels (standard vertical portrait aspect ratio for Pokémon TCG booster pack foil artwork).
* **Aspect Ratio:** `256:395` (Portrait).
* **File Format:** PNG.
* **Naming:** Use lowercase letters, numbers, and underscores (e.g., `xy_breakthrough_booster.png`). The filename (without extension) becomes the logical asset ID.

*Note: The compilation tool will automatically resize any non-standard PNGs to fit the proper template slot dimensions on server boot.*
